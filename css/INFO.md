## INFORMACIÓN SOBRE EL APARTADO DE CSS

Dentro de este apartado tenemos las hojas de estilo. En un principio, si todo tiene el mismo estilo nos quedaremos con un único "styles.css". Si decidisiemos hacer cambios de estilo en cada una de las diferentes páginas, el archivo "styles.css" pasará a llamarse "global.css" (para estilos globalizados y cosas generales) y a cada estilo le creamos su carpeta "style_1, style_2,..." y su respectivo archivo "styles.css"

Más adelante veremos cuántas páginas hacemos, cómo nos dividemos cada página y demás.
