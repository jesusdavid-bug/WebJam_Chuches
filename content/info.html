DECISIÓN DE LA TEMÁTICA

Elegimos el estilo neobrutalista como temática visual para nuestra web porque queríamos crear una experiencia impactante y memorable que destacara entre los demás proyectos. El neobrutalismo combina elementos visuales audaces como bordes gruesos, sombras duras y desplazadas, colores vibrantes y formas geométricas simples, creando una estética que evoca la nostalgia de los años 90 pero con un enfoque completamente moderno.

Este estilo nos permitió implementar elementos visuales distintivos como las sombras múltiples en el título principal "CHUCHES", los chips circulares con bordes gruesos en la navegación de colores, las tarjetas con efectos 3D mediante box-shadow, y los botones con desplazamiento al hacer hover. La combinación de estos elementos crea una identidad visual única y coherente en toda la web, haciendo que cada sección sea reconocible y mantenga la estética neobrutalista desde la página principal hasta las subpáginas de información.

ORGANIZACIÓN DEL TRABAJO

El proyecto se organizó de manera colaborativa, dividiendo las tareas según las fortalezas de cada miembro del grupo. Utilizamos una estructura modular donde cada persona se encargó de secciones específicas, pero manteniendo una comunicación constante para asegurar la coherencia visual y funcional en toda la web.

  • Planificación inicial: Definimos la estructura general, paleta de colores y estilo visual.
  • Desarrollo paralelo: Cada miembro trabajó en sus secciones asignadas simultáneamente.
  • Revisión conjunta: Realizamos revisiones periódicas para mantener la consistencia del diseño.
  • Integración final: Unificamos todos los componentes y ajustamos los detalles finales.

ESTRUCTURA DEL PROYECTO

  • Página principal (index.html): Hero section con navegación por colores y slider de equipo.
  • Secciones de sabores: Cada color tiene su propia sección con diseño neobrutalista.
  • Subpáginas de información: Páginas detalladas para cada sabor (rojo.html, naranja.html, etc.).
  • Página de información del proyecto: Esta página con los detalles del trabajo realizado.
  • Estilos CSS: Archivo centralizado (styles.css) con diseño neobrutalista y variables CSS.

TECNOLOGÍAS UTILIZADAS

  • HTML5: Estructura semántica y accesible del contenido.
  • CSS3: Estilos avanzados con variables CSS, grid, flexbox y animaciones.
  • Diseño neobrutalista: Bordes gruesos, sombras duras y colores vibrantes.
  • Sin JavaScript: Slider funcional usando solo CSS (inputs radio + labels).
  • Responsive design: Adaptación a diferentes tamaños de pantalla.
